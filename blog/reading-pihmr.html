PI-HMR: Towards Robust In-bed Temporal Human Shape Reconstruction with Contact Pressure Sensing

发表: CVPR 2025 (CCF-A) | arXiv: 2503.00068 单位: 中国科学技术大学（精密普适实验室，PPLab） 一作: Ziyu Wu*, Yufan Xiong*（共一）| 通讯: Xiaohui Cai

一、研究问题

用压力感知床单（pressure-sensing bedsheet）重建床上人体的 3D 形状与运动，替代传统视觉方案。

三大瓶颈:

  1. 压力数据特性未充分利用：单通道、零值背景无语义、压力轮廓与人体形状存在信息歧义
  2. 数据多样性差：压力数据集规模小（<20 人），传感器规格不一，难以整合
  3. 3D 标签质量差：单目 SMPLify 优化存在深度歧义（四肢误判悬空）和自穿透

二、整体框架（三件套工具链）

SMPLify-IB (标签生成) → PI-HMR (核心推理) → KD + TTO (性能增强)
组件 作用 核心技术
SMPLify-IB 为 TIP 数据集重新生成高质量 SMPL 伪标签 重力约束 + 轻量自穿透检测
PI-HMR 从压力序列回归 3D 人体网格 多尺度特征融合 (MFF)
跨模态 KD 将 RGB 域的人体先验迁移至压力域 CLIFF→HMR 特征图+输出蒸馏
测试时优化 (TTO) 用习惯先验修正推理结果 VQ-VAE 运动先验

为什么需要 SMPLify-IB？ 真值 GT 需要动捕设备，成本极高。已有的 TIP 数据集用普通 SMPLify 生成伪标签，质量差。SMPLify-IB 利用物理约束以算力换人力，生成更可靠标签作为训练监督信号。

三、SMPLify-IB：标签生成

3.1 核心创新

重力约束（解决深度歧义）

洞察: 人躺在床上静止时，四肢不应悬空（否则肌肉持续发力，不自然）。

  • 用 2D 关键点 GT 计算速度：若四肢速度 < 阈值且高度 > 床面 → 判定为单目歧义导致的"误判抬起"
  • 指数惩罚项将静态四肢推向床面：Lg = Σ I(v < thre) · exp(w·z)

轻量自穿透检测（解决自穿模）

对比传统: SMPLify-XMC 用 Winding Numbers 计算全部 6890 顶点 × 13776 面的实体角，需 22.62s/迭代。

新方法: 将 SMPL 网格聚类为 24 个近似凸的 segment，用向量点积判断穿透：

vi vj · vi cj  < 0  →  顶点 vi 在 segment Sj 内部 → 穿透
              ≥ 0  →  在外部

性能: 检测速度提升 53.9 倍（0.42s），准确率 98.32%。下采样 1/3 版本加速 450 倍。

双向力设计（既防穿透又促接触）

状态 检测 损失 作用
穿透 D(Si, Sj) > 0 L_push = + D
靠近但未穿透 SDF∈(0, 0.02m) L_pull = - C

距离衰减 exp(-λ·||ci - cj||) 确保只有相近的 segment 受作用——相当于类分子间作用力的势场。

3.2 优化目标

L_IB = λJ·LJ + λp·Lp + λsm·Lsm + λcons·Lcons + λbc·Lbc + λg·Lg + λsc·Lsc
损失 作用
LJ 2D 重投影误差（透视投影 + 标定焦距）
Lp 姿态 GMM 先验 + 形状正则 + 躯干合理高度（臀<肩，腰<均值）
Lsm 帧间速度/加速度平滑，消除抖动
Lcons 滑动窗口重叠帧的一致性
Lbc 床面接触：近床鼓励，穿床惩罚
Lg 重力约束
Lsc 自接触（穿透斥力 + 近邻引力）

3.3 两阶段优化

  • 第一阶段: 128 帧滑动窗口（50% 重叠），CLIFF 预测作初始化，联合优化 β/θ/t，仅处理手-躯干穿透
  • 第二阶段: 固定 β 为同一人所有帧的均值，仅优化 θ/t，处理全部身体 segment 间穿透
  • Adam, lr=0.01, 各 500 次迭代

四、PI-HMR：核心推理网络

4.1 整体架构

压力序列 V [T=15, H=64, W=64]
    │
    ▼
ResNet50 静态编码器（跨模态 KD 预训练权重）
    │  逐帧提取 → X = {x_t ∈ R^(2048×H₁×W₁)}
    ▼
多尺度特征融合模块 MFF
    │  → G = {g_t ∈ R^1024}
    ▼
2 层 Transformer（8 头，dropout 0.1，DropPath 0.2）
    │  时序建模 → 取均值作为中帧表示
    ▼
IEF SMPL 回归器
    │  → θ(24×3), β(10), t(3)
    ▼
SMPL 模型 → 3D mesh + 3D joints

4.2 MFF 多尺度特征融合（核心创新）

三条并行分支，Concat 融合为 1024 维特征：

ResNet50 输出 x_t
    │
    ├── ① Global Branch [512-dim]
    │     AttentionPooling（来自 CLIP）→ 全局上下文摘要
    │
    ├── ② Local Branch [256-dim]
    │     可学习 mask + 空间位置编码（传感器固定位置先验）→ AttentionPooling
    │
    └── ③ Sampling Branch [256-dim]
          TopK(压力值, K=128) 高压点 + 辅助 2D 关键点回归器预测的 12 个关节点
          → 在 upsampled 特征图上取特征 → 1 层 Transformer → 均值池化

SF-K 推理时的关键点来源：PI-HMR 内含一个辅助 2D 关键点回归器，从压力图预测 12 个 2D 关节点位置，受 L_2D 监督。推理时即用预测结果采样。

消融结果:

去掉组件 MPJPE
全部开启 57.13
去掉 SF-K 62.23（最伤）
去掉可学习 mask 60.95
去掉空间位置编码 60.65
去掉 AttentionPooling 59.21

TopK=128 最优（太少丢轮廓，太多引入噪声）。

4.3 IEF 回归器

Iterative Error Feedback，三轮迭代修正：

[z_final || θ_mean] → Δθ₀ → θ₁ = θ_mean + Δθ₀
[z_final || θ₁]     → Δθ₁ → θ₂ = θ₁ + Δθ₁
[z_final || θ₂]     → Δθ₂ → θ_final = θ₂ + Δθ₂

每次输入特征 z 和当前估计 θ 的拼接，预测修正残差 Δθ。本质是用网络模拟了梯度下降：不一次性跳到答案，而是由粗到精三步校正。β 和 t 直接由全连接层单次预测。

4.4 损失函数

L_pi = L_SMPL + 300·L_3D + 0.5·L_2D

L_SMPL = 1·||β-β̂||² + 60·||θ-θ̂||² + 60·||t-t̂||²
L_3D   = ||J_3D - Ĵ_3D||²
L_2D   = ||J_2D - Ĵ_2D||²（监督辅助 2D 关键点回归器）

训练：AdamW, lr=3e-4, 100 epochs, batch=16, 5 轮 warmup + cosine 调度, PyTorch 2.0.1, RTX 4090。

五、跨模态知识蒸馏

5.1 结构

教师: CLIFF (ResNet50 + IEF), 输入 RGB, 已在 COCO/MPII 等大规模数据预训练
学生: HMR  (ResNet50 + IEF), 输入压力图（单通道）
                   ↓ 共享架构 ↓
     4 个残差块的特征图维度天然对齐

5.2 三重监督

L_KD = L_pi(S(I_P), y)                    # GT 标签监督
     + L_pi(S(I_P), T(I_R))               # 教师输出蒸馏
     + Σ||M_S_i - M_T_i||                 # 4 层特征图 L1 对齐

依赖前提: TIP 数据集三模态同步采集——每帧同时有压力图、RGB、2D 标注，三元组天然存在。

消融:

配置 MPJPE
纯 GT 监督 75.06
GT + 输出蒸馏 77.86
GT + 特征蒸馏 67.34
三重全开 66.30

特征级蒸馏贡献最大——中间层的对齐传递了比输出标签更底层、更泛化的认知。

5.3 权重迁移

预训练后丢弃 IEF 回归器，仅将 ResNet50 编码器权重作为 PI-HMR 静态编码器的初始化，在端到端训练时继续微调。

六、测试时优化 (TTO)

6.1 动机

推理时没有 GT 供监督，也无法对压力图直接做 2D 关键点回归优化。但床上动作模式有限且规律——可以用预学的"习惯"修正。

6.2 机制

预训练 VQ-VAE 编码床上翻身的姿态序列模式（仅输入 θ，不含 β 和 t）：

PI-HMR 初始预测 θ₀ → VQ-VAE 重建 → 投影到最近的动作模式
                          ↓
L_TTO = α·L_m(θ, θ₀) + (1-α)·L_m(θ, VQ-VAE(θ)) + L_sm(θ)
  • 第一项约束不要偏离初始预测太远
  • 第二项拉向合理动作模式
  • 第三项平滑约束

30 次迭代，lr=0.01，滑动窗口 64，仅优化 θ。

八、局限与展望

  1. 手足精细度不足：SMPL 不支持手指/脚趾，无法精确刻画床上接触模式
  2. 未显式利用接触约束：端到端学习可能使预测的四肢脱离床面，需在模型中显式整合压力接触先验
  3. 信息歧义未完全解决：某些姿态下压力分布无法唯一确定人体姿态 → 期望引入大规模 VQ-VAE/Diffusion 运动生成模型

九、关键概念速查

概念 简述
SMPL 可微分人体参数化模型：θ(72维姿态) + β(10维体态) + t(平移) → 6890 顶点网格
IEF 迭代误差反馈：三轮残差修正，从平均姿态逐步逼近真值
HMR Kanazawa 2018 提出的单目端到端人体重建框架 (ResNet50 + IEF)
CLIFF ECCV 2022，HMR 改进版，额外输入全帧边界框位置和相机焦距，推断绝对位姿
伪标签 算法自动生成的弱标注，以算力换人力，桥接"标注不足"与"网络训练需求"
VQ-VAE 离散潜变量自编码器，将连续运动编码到码本中的离散 token，学习运动模式
